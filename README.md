# King's Analysis Process

## 1. Business Understanding
- 비즈니스에 대한 이해가 가장 중요하다.
### 1.1 Summary
- 해당 과제에 대한 주제, 배경, 목적 등을 철저히 이해한다.
### 1.2 Rule
- 해당 과제의 룰이 있다. 룰을 철저히 이해해야 삽질을 덜 한다.
### 1.3 Data Description
- 제공되는 테이블 정의서를 세밀하게 살핀다.
### 1.4 Discussion
- 다른 사람이 알아낸 정보를 꾸준히 습득하고 기록해둔다. 많은 도움이 된다.
